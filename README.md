# Smart-Menu

### This application aims to automate customer's order in the restaurant. It's a web application. Customer will get the facility to order directly without waiter. Order dashboard will show the order in the kitchen and will be processed further.

### Applicaion Functionalites
- Authentication
- Item and Category creation
- Viewing items for order
- Place order and Get Real time order processing status
- Payment for cloud kitchen

### Tech Stack
- Next Js
- Spring Boot
- PostgresSQL
- Liquibase
- Data Jpa
- Hibernate
- Hibernate Envers
